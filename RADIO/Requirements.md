# 요구사항

1. 토스트가 오른쪽 아래 또는 상단에 뜬다.

## 기능적 정의사항

1. 토스트가 어떨 때 오른쪽에 뜨고 어떨 때 상단에 떠야 하는가.
2. 토스트가 뜨는데는 몇 ms가 걸릴 것인가.
3. 토스트가 몇 초간 떠있는 상태로 유지될 것인가.
4. 토스트가 내려가는데는 몇 ms가 걸릴 것인가.
5. 토스트가 내려가는데는 나타난 방향의 반대로만 내려갈 것인가?
6. 최대 몇 개의 토스트가 뜰 수 있을 것인가?
7. 토스트는 사용자가 강제로 닫을 수 있게 할 것인가?

## 비기능적 정의사항

1. 접근성 처리는 어떻게 할 것인가?
2. 어떤 UI를 제공할 것인가?
3. 성능을 위해 어떤 CSS 속성을 사용할 것인가?
4. 토스트의 종류는 어떤 기준으로 나눌 것인가?
